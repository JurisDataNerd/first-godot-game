# 🎮 My First Godot Game

This project is my first game created using the **Godot Engine**.
It was developed as a learning project to understand the basics of game development, including player movement, level design, and simple gameplay mechanics.

## 🕹️ About the Game

This is a simple **2D platformer game** where the player controls a character to explore levels, jump over obstacles, and reach the goal.

The purpose of this project is to learn:

* Godot Engine fundamentals
* Scene and node system
* Basic scripting with GDScript
* Level design basics

## ⚙️ Built With

* **Godot Engine**
* **GDScript**

## 🚀 Running the Project

1. Install **Godot Engine**
2. Clone this repository

```bash
git clone https://github.com/yourusername/your-repository-name.git
```

3. Open **Godot**
4. Click **Import Project**
5. Select this project folder
6. Run the main scene
